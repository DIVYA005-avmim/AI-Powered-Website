# 🚀 AI-Powered Website

An AI-assisted website built by combining **Flow AI**, **Google AI Studio**, and **ChatGPT**. This project demonstrates how multiple AI platforms can be used together to accelerate the web development process—from idea generation and prompt engineering to design refinement and deployment.

## 💭 Reflection

AI is evolving rapidly, and it's natural to wonder how it will shape the future of work. Instead of focusing only on that uncertainty, I chose to explore it by building this project.

Working with Flow AI, Google AI Studio, and ChatGPT showed me that AI is more than a code generator—it's a learning partner. It helped me move from ideas to implementation much faster, while reminding me that creativity, critical thinking, and problem-solving remain essential.

This project reflects my journey of learning **with AI instead of fearing AI**.
